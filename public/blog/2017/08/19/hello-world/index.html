数理统计与数据分析

发表于 2017-08-19

第二章：随机变量

样本空间(sample space):所有可能的实验结果全体。记为$\Omega$ ,其中元素记为$\omega$.

事件(event):$\Omega$的子集。

随机变量(random variable): 从样本空间到实数域上的函数，记为$X$.

概率质量函数(probability mass function):$X$各种取值的概率，$p(x_i) = P(X = x_i)$和$\sum\limits_i{p(x_x)} = 1$ 的函数$p$.

累积分布函数(cumulative distribution function): $F(x) = P(X \leq x), -\infty < x < +\infty$

离散随机变量

伯努利随机变量：$p(x) = \begin{cases}p^x(1-p)^{1-x},&\text{若$x=0$或$x=1$}\0,&\text{否则}.\end{cases}$

示性随机变量(indicator random variable):如果$A$是一个事件，$I_A$在$A$发生时取1，不发生时取0.

二项分布：抛硬币n次,向上概率为$p$,正面向上的次数为随机变量$X$ $p(k) = \begin{equation} \left( \begin{array}{ccc} n\ k\ \end{array} \right) \end{equation}p^k(1-p)^{n-k}$

几何分布(geometric distribution):$X$表示做实验做到第一次成功的次数， $p(k) = (1-p)^{k-1}p$

负二项分布：$X$表示做实验直到成功$r$次为止所用的实验次数 $P(X = k) = \begin{equation} \left( \begin{array}{ccc} k-1\ r-1\ \end{array} \right) \end{equation}p^r(1-p)^{k-r}$

超几何分布(hypergeometric distribution):盒中$n$个球，其中$r$个黑球，$n-r$个白球，从中取出$m$个球，$X$表示抽到的黑球个数。 $p(X = k) = \frac{ \begin{equation} \left( \begin{array}{ccc} r\ k\ \end{array} \right) \left( \begin{array}{ccc} n-r\ m-k\ \end{array} \right) \end{equation}}{ \begin{equation} \left( \begin{array}{ccc} n\ m\ \end{array} \right) \end{equation}}$

泊松频率函数(Poisson frequency function):二项分布中$n$趋于无穷，$p$趋于零，$np = \lambda$时，频率函数为$p(k) = \frac{\lambda^ke^{-\lambda}}{k!}$

连续随机变量

均匀密度(uniform density): $U(a,b)$,$f(x) = \begin{cases}1,&a \leq x\n,&\text{$xb1$}.\end{cases}$

指数密度：$f(x) = \begin{cases}\lambda e^{-\lambda x},&x\geq0 \0,&x<0\end{cases}$电子元件任意时刻失效率相同，其寿命为$X$

韦伯分布：$f(x) = \begin{cases}\lambda ax^{a-1}e^{-\lambda x^a},&x\geq0 \0,&x<0\end{cases}$考虑老化，失效率随时间$x$上升$nx^m$,则$\lambda = n/(m+1)$

伽马密度：$g(t) = \frac{\lambda ^\alpha }{\Gamma(\alpha)}t^{\alpha-1}e^{-\lambda t}, t\geq0$

正态分布：$N(\mu,\sigma^2) $, $f(x) = \frac{1}{\sigma \sqrt{2\pi}}e^{\frac{-(x-\mu)^2}{2\sigma^2}}$

贝塔密度：$f(\mu) = \frac{\Gamma(a+b)}{\Gamma(a) \Gamma(b)}\mu ^{a-1}(1-\mu)^{b-1}$

第四章 期望与方差

矩：随机变量的$n$阶矩$mn = \int{-\infty}^{\infty}x^nf(x)dx$

矩生成函数，矩母函数(moment-generating function,mgf):$M(t) = E(e^{tX}) M(t) = \sum\limits_{n=0}\limits^{\infty}m_n \frac{t^n}{n!}$

如果矩生成函数在包含零点的开区间上存在，那么它唯一决定其概率分布。

第六章 正态分布的导出分布

$\chi^2$分布:如果$Z$是标准正态随机变量，$U = Z^2$的分布被称为自由度为1的卡方分布$\chi_1^2$，如果$U_1,U_2,\ldots,U_n$是相互独立的自由度为1的卡方随机变量，那么$V = U_1+U_2+\ldots+U_n$称为自由度为$n$的卡方分布，记作$\chi_n^2$。$E(V) = n, Var(V) = 2n$

$\chi_1^2$为$a = 1/2，\lambda = 1/2$的伽马分布，$\chi_n^2$为$a = n/2，\lambda = 1/2$的卡方分布。

相同$\lambda$值的独立伽马随机变量之后服从伽马分布，所以如果$U$和$V$独立，$U\sim\chi_n^2,V\sim\chim^2$,那么$U+V\sim\chi{n+m}^2$

t分布(t distribution):如果$Z\sim N(0,1), U\sim \chi_n^2$,且$Z$和$U$独立，那么$Z/\sqrt{U/n}$是自由度为$n$的t分布。

样本均值和样本方差:令$X_1,\ldots,X_n$是独立的$N(\mu,\sigma^2)$随机变量，我们有时称之为来自正态分布的样本(sample)

样本均值为$\bar{X} = \frac{1}{n} \sum\limits_{i=1}\limits^{n}X_i$

样本方差为$S^2 = \frac{1}{n-1}\sum\limits_{i=1}\limits^n(X_i-\bar{x})^2$

第七章 抽样调查

总体参数

假设总体大小是$N$,总体成员的特征数值记为$x_1,x_2,\ldots,x_n$,可以是年龄、体重或者性别这种离散变量。

如果N=393表示医院的总数，$xi$表示第$i$个医院出院的人数，总体均值为$\mu = \frac{1}{N}\sum\limits{i=1}\limits^Nxi$,总体总数$\tau = N\mu$,总体方差为$\sigma^2 = \frac{1}{N}\sum\limits{i=1}\limits^N(x_i-\mu)$

简单随机抽样(simple random sampling,s.r.s)

任何容量为$n$的样本都具有相同的发生概率，也就是说无重复抽取容量为$n$的$ \begin{equation} \left( \begin{array}{ccc} N\ n\ \end{array} \right) \end{equation}$种可能的样本，每个都具有相同的概率。

n表示样本容量(n小于N),样本均值为$\bar{X} = \frac{1}{n}\sum\limits_{i=1}^nX_i$
